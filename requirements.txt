Requirements

No external applications required to run this simulation.

The development of the code was done in a local environment with:
MATLAB R2024a update 3 & MATLAB Parallel computing toolbox

The testing of the code was done in Aalto's Triton cluster with:
MATLAB r2023b & MATLAB Parallel computing toolbox

For the hardware specifications, please refer to the report that is available in the documentation of this repository.
